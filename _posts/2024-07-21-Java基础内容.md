### 2.8 if…else,switch,for综合应用实例

~~~java
import java.util.Scanner;
public class instance {
    //输入一个年份和月份，输出本月的日历
    public static void main(String[] args) {
        //1、接受收入的年份和月份
        Scanner input = new Scanner(System.in);
        System.out.println("Please input a year:");
        int year = input.nextInt();
        System.out.println("Please input a month:");
        int month = input.nextInt();
        //2、计算这个年份和月份第一天对应的是星期几
        //%7   1900年1.1是星期一    总天数%7
        int days = 0;//记录总天数
        //循环年份
        for (int i = 1990;i < year;i++){
            //判断闰年来记录天数
            if (i % 4 == 0 && i % 100 != 0 || i % 400 == 0){
                days += 366;
            }else
                days += 365;
        }
        //循环月份
        for (int i = 1; i < month; i++){
            if (i == 2){
                //平年28，闰年29
                if (year % 4 == 0 && year % 100 != 0 || year % 400 == 0){
                    days += 29;
                }else {
                    days += 28;
                }
            }else if(i == 4 || i == 6 || i == 9 || i == 11){
                days += 30;
            }else
                days += 31;
        }
        //计算输入月份的一号是星期几
        int weekday = 1 + days % 7;
        System.out.println("First day is weekday: " + weekday);

        //3、打印空格，日期，换行
        //3.1 打印表头
        System.out.println("一\t二\t三\t四\t五\t六\t日");
        //3.2 打印空格
        for (int s = 1; s < weekday; s++){
            System.out.print('\t');
        }
        //3.3 打印日期
        //首先要知道这个月有多少天
        int day;
        switch (month){
            case 2:
                if (year % 4 == 0 && year % 100 != 0 || year % 400 == 0){
                    day = 29;
                }else
                    day = 28;
                break;
            case 4:
            case 6:
            case 9:
            case 12:
                day = 30;break;
            default:
                day = 31;break;
        }
        //打印日期
        for (int j = 1; j <= day; j++){
            System.out.print(j + "\t");
            if ((weekday - 1 + j) % 7 == 0){
                System.out.println();
            }
        }
    }
}
~~~

### 2.9 方法

方法（其他语言中的函数，在面向对象编程中叫方法）包含一个方法头和一个方法体。下面是一个方法的所有部分：

- **修饰符：**修饰符，这是可选的，告诉编译器如何调用该方法。定义了该方法的访问类型。
- **返回值类型 ：**方法可能会返回值。return Value Type 是方法返回值的数据类型。有些方法执行所需的操作，但没有返回值。在这种情况下，return Value Type 是关键字**void**。
- **方法名：**是方法的实际名称。方法名和参数表共同构成方法签名。
- **参数类型：**参数像是一个占位符。当方法被调用时，传递值给参数。这个值被称为实参或变量。参数列表是指方法的参数类型、顺序和参数的个数。参数是可选的，方法可以不包含任何参数。
- **方法体：**方法体包含具体的语句，定义该方法的功能。

    public static void bubbleSort(int[] nums) {
        //外层循环控制轮数
        for (int i = 0; i < nums.length - 1; i++) {
            //内层虚幻控制每轮比较的次数
            for (int j = 0; j < nums.length - i - 1; j++) {
                //比较相邻，交换位置
                if (nums[j] > nums[j + 1]) {
                    int temp = nums[j];
                    nums[j] = nums[j + 1];
                    nums[j + 1] = temp;
                }
            }
        }
    }
}

## 3. 面向对象

#### 3.1 对象和类的概念

* 面向对象思想
  * 一切客观存在的食物都是对象，<u>**万物皆对象**</u>。
  * 任何对象一定有自己的特征（称为属性）和行为（称为方法）。

* 类可以认为是一个蓝图（类型），对象可以认为是根据蓝图的实例
* 类：定义了对象应具有的特征和行为，类是对象的模板。
* 对象：拥有多个特征和行为的实例，对象是类的实例。

#### 3.2 成员变量和成员方法

|          |        局部变量        |       实例变量       |
| :------: | :--------------------: | :------------------: |

|  默认值  |           无           | 字面值（与数组相同） |

#### 3.3 方法重载

* 重载：一个类中定义了多个相同名称的方法。
* 要求：
  * 方法名称相同
  * 参数列表不同（类型、个数、顺序）
  * 与访问修饰符、返回值类型无关

* 调用带有重载的方法是，需要根据传入的实参去找到与之匹配的方法
* 好处：灵活、方便、屏蔽使用差异

#### 3.4 构造方法

* 构造方法：类中特殊的方法，主要用于创建对象。
* 特点：
  * 名称鱼类名完全相同
  * 没有返回值类型
  * 创建对象时，触发构造方法的调用，不可通过点手动调用

* 如果在类中显式定义构造方法，则编译器默认提供无参构造方法。
* **对象的创建过程**：
  * 内存中开辟对象空间
  * 为各个属性赋予初始值
  * 执行构造方法中的代码
  * 将对象的地址赋给变量

**引用类型就是栈里面的地址指向了堆立面的空间**。

* 构造方法也可重载，遵循·重载规则

**注意：如果有带参构造就必须手动加上无参构造。**
